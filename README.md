# video-sampler
Extract screenshots from a video at regular intervals (seconds/frames) or specific timestamps.

Run `video_sampler_ui.ipynb` to configure and execute.

Tested with Python 3.10

**Dependencies**
```
opencv-python
Pillow
```

**To Install Dependencies**
```
pip install -r requirements.txt
```
